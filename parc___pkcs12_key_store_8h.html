PARCPkcs12KeyStore* parcPkcs12KeyStore_Acquire ( const PARCPkcs12KeyStore *  instance)

Increase the number of references to a PARCPublicKeySigner instance.

Note that new PARCPublicKeySigner is not created, only that the given PARCPublicKeySigner reference count is incremented. Discard the reference by invoking parcPublicKeySigner_Release.

Parameters
[in]instanceA pointer to a valid PARCPublicKeySigner instance.
Returns
The same value as instance.

Creates a PKCS12 keystore identity with a self-signed certifiate.

Note that this call currently aborts if keystore i/o access fails, behavior that may change in the future.

Parameters
[in]filenameThe name of the PKCS12 file.
[in]passwordThe password to open the PKCS12 file.
[in]subjectNameThe certificate subject associated with the PKCS12 file.
[in]keyLengthThe length of the public key associated with the PKCS12 file.
[in]validityDaysThe validity (in days) of the certificate associated with the PKCS12 file.
Returns
true on success, false if certificate creation fails, and will abort if keystore i/o fails.

Example:

1 {
2  const char *filename = "/tmp/ccnxFileKeyStore_Pkcs12Open_CreateAndOpen.p12";
3  const char *password = "12345";
4  const char *subject = "alice";
5  bool result = parcPkcs12KeyStore_CreateFile(filename, password, subject, 1024, 32);
6 }

void parcPkcs12KeyStore_Release ( PARCPkcs12KeyStore **  instancePtr)

Release a previously acquired reference to the given PARCPkcs12KeyStore instance, decrementing the reference count for the instance.

The pointer to the instance is set to NULL as a side-effect of this function.

If the invocation causes the last reference to the instance to be released, the instance is deallocated and the instance's implementation will perform additional cleanup and release other privately held references.

Parameters
[in,out]instancePtrA pointer to a pointer to the instance to release.

Example:

1 {
2  PARCPkcs12KeyStore *a = parcPkcs12Store_Open(...);
3 
4  parcPkcs12KeyStore_Release(&a);
5 }
