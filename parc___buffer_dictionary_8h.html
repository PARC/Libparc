Detailed Description

A key/value dictionary built around PARCBuffer as the Key and the Value.

The key/value dictionary models the Java MAP interface. It is built around Put, Get, and Remove. The dictionary stores references to the Key and Value, so the caller may destroy its references if no longer needed.

Add a key/value to the dictionary, returns previous value or NULL.

The Dictionary will store a reference (PARCBuffer::aquire) to the key and to the value. The Key and Value must be non-NULL. If a previous entry for the key is in the dictionary, the previous value is returned. THE CALLER MUST USE parcBuffer_Release on the returned value if it is non-NULL;
