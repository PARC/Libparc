Detailed Description

In-memory cache of keys or certificates.

Not sure how to differentiate between keys and certs at the moment. The current API is thus built around keys.

Adds the specified key to the keycache.

Parameters must be non-null Returns true if added or false if keyid alredy existed and was a different than key This will store its own reference to the key, so the caller must free key.

Parameters
[in]cacheA pointer to a PARCCryptoCache instance.

Example:

1 {
2  PARCKey *key = ....;
3  PARCCryptoCache *cache = parcCryptoCache_Create();
4  parcCryptoCache_AddKey(cache, key);
5  parcKey_release(&key);
6  // do stuff with the crypto cache
7  parcCryptoCache_Destroy(&cache);
8 }
