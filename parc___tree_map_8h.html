A Red-Black tree containing PARCObject keys and values. More...

Detailed Description

A Red-Black tree containing PARCObject keys and values.

The map is sorted according to the natural ordering of its keys, or by a comparator function provided at creation time, depending on which constructor is used.

typedef int( PARCTreeMap_CustomCompare) (const PARCObject *key1, const PARCObject *key2)

Definition of a custom funcion to compare two keys.

A function of this signature can be provided to CreateCustom(...) constructor to override the default parcObject_Compare(...) for comparing key objects. This will be used during all internal comparisons.

Parameters
[in]key1The first key to compare
[in]key2The second key to compare
Returns
A signum comparison. negative if key1 is smaller than key2, 0 if equal, positive if key1 is bigger.

Example:

1 {
2  int _compareKeys(const PARCObject *key1, const PARCObject *key2) {...}
3 
4  PARCTreeMap * tree1 = parcTreeMap_CreateCustom(_compareKeys);
5 
6  ...
7 
8  parcTreeMap_Release(&tree1);
9 }

Get a PARCList of the keys from a PARCTreeMap.

All keys will be valid keys in the PARCTreeMap. The caller will own the list of keys and should release it when done. The caller will not own the keys themselves. Note that if the tree is modified or destroyed the key pointers might no longer point to valid keys. The list of keys will be sorted as per the provided or key's default compare function.

Parameters
[in]treeA pointer to a PARCTreeMap.

A list of (pointers to) keys.

PARCObject* parcTreeMap_Remove ( PARCTreeMap *  tree,
const PARCObject *  key 
)

Remove an entry from a PARCTreeMap.

The entry will be removed from the tree and the tree's reference to the key will be released. The value will be returned and responsibility for releasing the reference to the value will transfer to the caller. The provided key will not be modified. If the key is not found in the tree, NULL is returned.

Parameters
[in,out]treeA pointer to an initialized PARCTreeMap.
[in]keyA pointer to a key (The tree will not own this).
Returns
A pointer to a value or NULL. You will now own this value.

2  PARCTreeMap * tree1 = parcTreeMap_Create(.....);

4  ...
