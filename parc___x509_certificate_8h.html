PARCX509Certificate* parcX509Certificate_Acquire ( const PARCX509Certificate *  certificate)

Increase the number of references to a PARCX509Certificate instance.

Note that a new PARCX509Certificate is not created, only that the given PARCX509Certificate reference count is incremented. Discard the reference by invoking parcX509Certificate_Release.

Parameters
[in]certificateA pointer to the original instance.
Returns
The value of the input parameter instance.

Example:

1 {
2  PARCX509Certificate *x = parcX509Certificate_CreateFromPEMFile(...);
3  PARCX509Certificate *x2 = parcX509Certificate_Acquire(x);
4 
5  parcX509Certificate_Release(&x);
6  parcX509Certificate_Release(&x2);
7 }

void parcX509Certificate_Release ( PARCX509Certificate **  certificateP)

Release a previously acquired reference to the specified instance, decrementing the reference count for the instance.

The pointer to the instance is set to NULL as a side-effect of this function.

If the invocation causes the last reference to the instance to be released, the instance is deallocated and the instance's implementation will perform additional cleanup and release other privately held references.

Parameters
[in,out]certificatePA pointer to a pointer to the instance to release.

Example:

1 {
2  PARCX509Certificate *x = parcX509Certificate_CreateFromPEMFile(...);
3 
4  parcX509Certificate_Release(&x);
5 }
