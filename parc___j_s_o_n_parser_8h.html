PARCJSONParser* parcJSONParser_Acquire ( const PARCJSONParser *  parser)

Increase the number of references to a PARCJSONParser.

Note that new PARCJSONParser is not created, only that the given PARCJSONParser reference count is incremented. Discard the reference by invoking parcJSONParser_Release.

Parameters
parserA pointer to the original instance.
Returns
The value of the input parameter parser.

Example:

1 {
2  PARCBuffer *buffer = parcBuffer_WrapCString(" { \"name\" : 123 }");
3 
4  PARCJSONParser *parser = parcJSONParser_Create(buffer);
5  PARCJSONParser *x2 = parcJSONParser_Acquire(parser);
6 
7  parcJSONParser_Release(&parser);
8  parcJSONParser_Release(&x2);
9 }
See also
parcJSONParser_Release

void parcJSONParser_Release ( PARCJSONParser **  parserPtr)

Release a previously acquired reference to the specified instance, decrementing the reference count for the instance.

The pointer to the instance is set to NULL as a side-effect of this function.

If the invocation causes the last reference to the instance to be released, the instance is deallocated and the instance's implementation will perform additional cleanup and release other privately held references.

The contents of the dealloced memory used for the PARC object are undefined. Do not reference the object after the last release.

Parameters
[in,out]parserPtrA pointer to a pointer to the instance of PARCJSONParser to release.

Example:

1 {
2  PARCBuffer *buffer = parcBuffer_WrapCString(" { \"name\" : 123 }");
3 
4  PARCJSONParser *parser = parcJSONParser_Create(buffer);
5 
6  parcJSONParser_Release(&parser);
7 }
