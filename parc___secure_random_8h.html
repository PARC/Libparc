PARCSecureRandom* parcSecureRandom_Acquire ( const PARCSecureRandom *  instance)

Increase the number of references to a PARCSecureRandom instance.

Note that new PARCSecureRandom is not created, only that the given PARCSecureRandom reference count is incremented. Discard the reference by invoking parcSecureRandom_Release.

Parameters
[in]instanceA pointer to a valid PARCSecureRandom instance.
Returns
The same value as instance.

Example:

1 {
2  PARCSecureRandom *a = parcSecureRandom_Create();
3 
4  PARCSecureRandom *b = parcSecureRandom_Acquire();
5 
6  parcSecureRandom_Release(&a);
7  parcSecureRandom_Release(&b);
8 }

void parcSecureRandom_Release ( PARCSecureRandom **  instancePtr)

Release a previously acquired reference to the given PARCSecureRandom instance, decrementing the reference count for the instance.

The pointer to the instance is set to NULL as a side-effect of this function.

If the invocation causes the last reference to the instance to be released, the instance is deallocated and the instance's implementation will perform additional cleanup and release other privately held references.

Parameters
[in,out]instancePtrA pointer to a pointer to the instance to release.

Example:

1 {
2  PARCSecureRandom *rng = parcSecureRandom_Create();
3 
4  parcSecureRandom_Release(&rng);
5 }
