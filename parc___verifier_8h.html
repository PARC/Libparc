void parcVerifier_Release ( PARCVerifier **  verifierPtr)

Release a previously acquired reference to the specified instance, decrementing the reference count for the instance.

The pointer to the instance is set to NULL as a side-effect of this function.

If the invocation causes the last reference to the instance to be released, the instance is deallocated and the instance's implementation will perform additional cleanup and release other privately held references.

The contents of the dealloced memory used for the PARC object are undefined. Do not reference the object after the last release.

Parameters
[in,out]verifierPtrA pointer to a pointer to the instance to release.

Example:

1 {
2  PARCVerifier *verifier = parcVerifier_Create(verifierInstance, PARCInMemoryVerifierAsVerifier);
3 
4  parcVerifier_Release(&verifier);
5 }

Verify the signature against the provided digest with the specified key.

If we do not trust the key, the signature will be rejected. In this context, trusting a key means that it was previously added to this verifiers "store".

Returns true if the signature is accepted,false if it is rejected.

Parameters
[in]verifierA PARCVerifier instance.
[in]keyIdA PARCKeyId which identifies the verification key.
[in]hashDigestA PARCCryptoHash which stores the locally computed digest.
[in]suiteThe PARCCryptoSuite in which verification is performed.
[in]signatureThe PARCSignature which is to be verified.
Return values
trueIf the signature is valid
falseOtherwise
