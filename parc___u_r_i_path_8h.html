As strings, URI paths are compared in normal lexographical order. This is analogous to strcmp(...).

Returns

bool parcURIPath_Equals ( const PARCURIPath *  pathA,
const PARCURIPath *  pathB 
)

Determine if two PARCURIPath instances are equal.

This function implements the following equivalence relations on non-null PARCURIPath instances:

  • It is reflexive: for any non-null reference value x, parcURIPath_Equals(x, x) must return true.
  • It is symmetric: for any non-null reference values x and y, parcURIPath_Equals(x, y) must return true if and only if parcURIPath_Equals(y x) returns true.
  • It is transitive: for any non-null reference values x, y, and z, if parcURIPath_Equals(x, y) returns true and parcURIPath_Equals(y, z) returns true, then parcURIPath_Equals(x, z) must return true.
  • It is consistent: for any non-null reference values x and y, multiple invocations of parcURIPath_Equals(x, y) consistently return true or consistently return false.
  • For any non-null reference value x, parcURIPath_Equals(x, NULL) must return false.
Parameters
[in]pathAFirst instance
[in]pathBSecond instance
Returns
true Equal PARCURIPath instances
false Otherwise

Example:

1 {
2  char *pointer;
3  PARCURIPath *pathA = parcURIPath_Parse("lci:/foo/, &pointer);
4  PARCURIPath *pathB = parcURIPath_Parse("lci:/foo/, &pointer);
5 
6  if (parcURIPath_Equals(pathA, pathB) {
7  printf("Paths are equal\n");
8  }
9 
10  parcURIPath_Release(&pathA);
11  parcURIPath_Release(&pathB);
12 }

void parcURIPath_Release ( PARCURIPath **  pathPtr)

Release a previously acquired reference to the specified instance, decrementing the reference count for the instance.

The pointer to the instance is set to NULL as a side-effect of this function.

If the invocation causes the last reference to the instance to be released, the instance is deallocated and the instance's implementation will perform additional cleanup and release other privately held references.

[in,out]pathPtrA pointer to a pointer to the instance to release.

Example:

3 
4  parcURIPath_Release(&auth);
5 }
