PARCCertificateFactory* parcCertificateFactory_Acquire ( const PARCCertificateFactory *  factory)

Increase the number of references to a PARCCertificateFactory instance.

Note that a new PARCCertificateFactory is not created, only that the given PARCCertificateFactory reference count is incremented. Discard the reference by invoking parcCertificateFactory_Release.

Parameters
[in]certificateA pointer to the original instance.
Returns
The value of the input parameter instance.

Example:

1 {
2  PARCCertificateFactory *x = parcCertificateFactory_CreateFromFile(...);
3  PARCCertificateFactory *x2 = parcCertificateFactory_Acquire(x);
4 
5  parcCertificateFactory_Release(&x);
6  parcCertificateFactory_Release(&x2);
7 }

void parcCertificateFactory_Release ( PARCCertificateFactory **  factoryP)

Release a previously acquired reference to the specified instance, decrementing the reference count for the instance.

The pointer to the instance is set to NULL as a side-effect of this function.

If the invocation causes the last reference to the instance to be released, the instance is deallocated and the instance's implementation will perform additional cleanup and release other privately held references.

Parameters
[in,out]factoryPA pointer to a pointer to the instance to release.

Example:

1 {
2  PARCCertificateFactory *x = parcCertificateFactory_CreateFromFile(...);
3 
4  parcCertificateFactory_Release(&x);
5 }
